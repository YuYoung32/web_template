# WebTemplate

自用go-gorm-gin-redis-mysql-jwtToken-restAPI web脚手架

* 支持配置文件
* 支持log（gin日志中间件+局内编程log）
* 支持mysql和redis
* 优雅关闭
* 支持jwtToken权限认证
* CORS中间件
